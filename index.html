Student Sleep Analysis

Analyzing the Impact of Sleep Patterns on Academic Performance

About the Project

This project analyzes how sleep duration affects students' GPA using Python, Pandas, NumPy, and Matplotlib.

Key Insights

  • Students sleeping 7–8 hours show better academic performance.
  • Very low sleep reduces productivity and GPA.
  • Balanced sleep improves concentration.

Technologies Used

Python | Pandas | NumPy | Matplotlib | Jupyter Notebook
